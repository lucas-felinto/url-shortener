## About
A small application developed to short urls. The intention of this project was improve my skills in NodeJS and JavaScript and learn about MongoDB.

## Features
* HTML, CSS and JavaScript
* NodeJS
* Express
* MongoDB

#### Dependences:
* Mongoose
* Nodemon
* ShortID
* Nunjucks
